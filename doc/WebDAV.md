### WebDAV 请求方法

- [ ] GET：检索文档
- [ ] HEAD：类似于 GET，用于检查资源是否存在，以及得到资源元数据
- [ ] PropFind：检索资源属性、目录层次结构
- [ ] POST：添加资源
- [ ] PUT：修改资源
- [ ] MKCOL：创建集合或目录
- [ ] COPY，MOVE：资源复制/移动
- [ ] DELETE：销毁资源或集合
- [ ] OPTIONS：询问可执行哪些方法
- [ ] TRACE：远程诊断服务器
- PropPatch：更改和删除资源属性 **（暂不支持）**
- Lock：锁定资源 **（暂不支持）**
- Unlock：解除资源锁定 **（暂不支持）**
